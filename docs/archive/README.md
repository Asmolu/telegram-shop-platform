# Архив документации

В эту папку перемещаются исторически полезные документы только после проверки inbound links.
Каждый archived document должен содержать дату deprecation, причину, canonical replacement и
предупреждение, что он не является текущей operational truth. На 2026-07-13 исторические ADRs
не перемещались и не удалялись.
